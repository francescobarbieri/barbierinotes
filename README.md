## Barbieri Notes

![banner](public/img/ogImage.png)

My current personal websites repository. Available at [barbierinotes.com](https://www.barbierinotes.com).

## Overview

- `/` - Home page.
- `/posts` - List of essays page. 
- `/posts/[slug]`- Single essay page, served with Astro collections of .mdx files.

## Built With

- Framework: [Astro.js](https://astro.build)
- Deployment: [Vercel](https://vercel.com)
- Styling: [Tailwind CSS](https://tailwindcss.com)
- Analytics: [Vercel Analytics](https://vercel.com)
